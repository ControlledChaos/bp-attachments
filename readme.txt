BP Attachments is a BuddyPress component to help others deal with attachments.

== Description ==

<h4>BuddyPress version</h4>

<strong>Requires at least BuddyPress 2.0-alpha-8020</strong>

<h4>More Information</h4>

Visit the <a href="https://buddypress.trac.wordpress.org/ticket/5429">ticket #5429</a> for more information about this BuddyPress component.

== Installation ==

Make sure BP Attachments is uploaded to "/wp-content/plugins/bp-attachments/".

Activate BP Attachments in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, make sure to activate BP Attachments at the same level than BuddyPress: if BuddyPress is network activated, then network activate BP Attachments, if BuddyPress is activated on a blog, then activate BP Attachments on the same blog.
